# node-auth-jwt

> ⚔️ Authentication with JSON Web Token (JWT).

## Install

Install dependencies
```
$ npm i
```
Create `.env` file
```
MONGODB_URI=<Your Mongodb URI>
```

## Usage

One line to run
```
$ npm start
```

## API

| Route            | HTTP method | Description                 |
|------------------|-------------|-----------------------------|
| /api/users       | `GET`       | Get all users.              |
| /api/authenticate| `POST`      | Authenticate account.       |
